# Runtime Baseline

These files are the captured live sprite shader baselines for the target runtime.

Provenance:

- `fpsprite.glsl` and `fpselect.glsl` were copied from a user-extracted BGEE `2.6.6.x` runtime on 2026-04-05.

They are stored here to keep future edits anchored to the real starting point instead of to external examples or memory.
